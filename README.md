# N2日语单词助手
一款纯前端离线日语单词背诵网页工具，基于 GitHub Pages 免费部署，无后端、无需联网登录，自定义词库自由导入，适配手机/电脑。

## 在线访问
https://nyakoo.github.io/

## 项目简介
本工具专为N2日语学习者打造，支持自定义 txt 单词词库，内置四种记忆模式，提供浏览学习、假名拼写、汉字默写、连连看配对功能。
全部代码为静态 HTML+JS，数据仅保存在浏览器本地，不会上传任何单词信息，隐私安全。

## 核心功能
### 1. 双种词库加载方式
1. **虚拟数字九宫格加载（在线Pages专用）**
   在仓库 `WordTxt` 文件夹存放 `1.txt`、`2.txt` 数字命名词库，页面输入数字一键读取对应单词本；
2. **本地文件上传（离线本地打开可用）**
   直接上传电脑内 txt 文件，无需部署也能使用。

### 2. 四大背诵模式
- **学习模式**：完整展示汉字、读音、声调、释义，支持边看边拼写输入自动切卡；
- **假名挑战**：给出汉字，输入假名/罗马音作答，答错自动标记难词延后重复复习；
- **书写测试**：给出中文释义，点击乱序字块拼出汉字，强化汉字记忆；
- **连线消除**：中日单词配对连连看，轻松巩固词汇。

### 3. 复习优化机制
- 答错/点击查看答案的单词自动收录「难词列表」；
- 通关后可选择：排除难词重刷 / 只复习难词专项巩固；
- 罗马音自动转假名输入，支持标准五十音罗马音拼写规则。

## 词库 txt 格式规范
- 每行一个单词，以半角逗号分隔，依次为单词、读音、声调、含义。编码必须为 **UTF-8**。
- 举例：毛糸,けいと,0,毛線
